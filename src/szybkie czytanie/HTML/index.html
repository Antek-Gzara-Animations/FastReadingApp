"Fast Reading" App - Help Page

Table of contents:



General

UI:
-"Paste from clipboard or write": open window with basic text editor.
-"start/stop" - start's playing text. If file is not selected or no text written the demo would be played.
-"open file" - select text file. WARNING: the file extention must be .txt!
-"hide" - hide application to taskbar.
-"exit" - closes application.



opening file

This button will open "select file" dialog window. Find your text file and click confirm. Then, press "start" button to play text animation.



How to uninstall?


In order to uninstall, you need to search in your windows task bar "add or remove programs", find the program, click uninstall.


How to update?

In order to update, you need to search in your windows task bar "add or remove programs", find the program, click uninstall. Then find the newest version and download the installer. Install the newest version of app.
